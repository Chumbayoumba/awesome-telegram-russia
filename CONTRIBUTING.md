# How to Contribute

We'd love your help making this list better!

## Adding a Resource

1. Fork this repository
2. Add your resource to the appropriate section
3. Keep descriptions concise and informative
4. Open a Pull Request

## Guidelines

- Check if the resource already exists
- Ensure the resource is relevant to Telegram development in Russia
- Follow alphabetical order within sections
- Use proper markdown formatting

## Questions?

Feel free to open an issue or contact [@longfest](https://t.me/longfest) on Telegram.
